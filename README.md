# My Proxy

A lightweight Node.js web proxy built for simple browser routing.

## Features

- Simple web proxy
- Lightweight and fast
- Easy deployment
- Built with Node.js

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR-USERNAME/my-proxy.git
cd my-proxy
```

Install dependencies:

```bash
npm install
```

## Run Locally

```bash
npm start
```

The proxy runs on:

```text
http://localhost:3000
```
